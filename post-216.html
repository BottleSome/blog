Minecraft服务器正版验证出问题？-Bottle Minecraft服务器正版验证出问题？ 20180713 ## 最近服务器经常有验证无法通过的情况，后台显示： ``` Couldn't verify username because servers are unavailable ``` ![cobcat桥](https://api.xbottle.top/i.php?u=https://wx4.sinaimg.cn/large/ed039e1fgy1ft8df678laj20nq0dc493) 这个问题的处理方式其实并不难，用Centos作演示： ``` vim /etc/hosts/ ``` 编辑hosts文件，加入： ``` 216.137.36.183 sessionserver.mojang.com ``` 即可将sessionserver.mojang.com直接指向其ip，免去了DNS无法解析的问题~ 实测这样之后**一定程度**上减少了验证问题. --------------------------------------- 要更高程度解决这种问题就要用到代理，但是目前我还没有琢磨出来这种方法，听Akkariin大佬说是要用到authlib... 咸鱼技术,Minecraft 216
